In this repo you will find all my personal config files. That I use for the tiling window manager swayfx.
You can copy all my files and use them for a better sway experience. 
That config files work for all Linux distors, that can run swayfx.

You will need to install the following packages:

swayfx
alacritty
wofi
fastfetch
waybar
nerd font
swayosd
grim
slurp
cliphist

I am sure, that I am missing some packages
